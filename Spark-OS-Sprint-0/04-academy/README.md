# Academy

The Academy answers:

> How do we teach the Canon and develop professional judgment?

Spark Academy is designed to minimize time to meaningful competence rather than maximize training hours.

## Educational Aim

The Academy should produce brokers who can:

- understand principles
- apply knowledge to unfamiliar situations
- recognize uncertainty
- make defensible decisions
- know when to use reference material
- know when escalation is required
- explain the reasoning behind an answer

## Relationship to the Canon

The Canon defines the accepted knowledge.

The Academy adds:

- sequence
- explanation
- examples
- scenarios
- application exercises
- assessments
- reflection
- field practice
- mentor discussion

Academy content should not become a competing source of truth.

## Planned Learning Layers

1. Learn
2. Apply
3. Master

## Planned Entry Structure

- Induction — how Spark learning works
- Mindset — how a Spark mortgage broker thinks
- Foundations
- Qualification
- Products
- Sales
- Operations

These structures remain provisional until the philosophical and educational governance documents are accepted.

## Assessment Direction

Assessments should distinguish between:

- recall
- understanding
- application
- professional judgment

High scores should require reasoning, not merely memorization.
