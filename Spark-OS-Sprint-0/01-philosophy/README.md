# Philosophy

The Philosophy area answers:

> What do we believe?

It contains the foundational beliefs that should guide the Canon, Academy, operations, AI behavior, and future software.

## Planned Structure

1. `manifesto.md` — why Spark OS and Spark Academy must exist
2. `values.md` — what Spark refuses to compromise
3. `principles.md` — durable rules derived from those values
4. `mental-models.md` — reusable ways of interpreting decisions and problems
5. `glossary.md` — precise definitions of foundational language

## Authority

Philosophy is more foundational than a course, procedure, software feature, or AI prompt.

When an output conflicts with accepted Philosophy, the output should be reconsidered.

## Current Status

The philosophical documents are intentionally not drafted in Sprint 0. Sprint 0 establishes the structure and governance required to develop them deliberately.
