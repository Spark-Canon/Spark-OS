# Roadmap

## Phase 0 — Project Memory

- Install Constitution
- Install current-state resume point
- Install domain language
- Install principles
- Install responsibility map
- Install ADR process
- Install architectural journal

## Phase 1 — Domain Foundation

- Define client, borrower, contact, application, mortgage file, and participant models
- Define identity and role boundaries
- Define audit requirements
- Define document model
- Define workflow primitives

## Phase 2 — Operational MVP

- Configurable milestones and steps
- Task assignment
- Document requests and tracking
- Borrower portal
- Internal mortgage-file workspace
- Basic notifications
- Audit history

## Phase 3 — Mortgage Processing

- Application data
- Income and liability inputs
- Underwriting calculations
- Conditions
- Lender submission packages
- Decision and exception tracking

## Phase 4 — Integrations

- Finmo/Lendesk
- Filogix or Velocity
- Gmail and calendar
- CRM imports
- Document storage
- Builder and partner workflows

## Phase 5 — Intelligence

- AI-assisted file summaries
- Missing-document detection
- Workflow recommendations
- Underwriting assistance
- Communication drafting
- Operational risk alerts

## Phase 6 — Brokerage Platform

- Training and onboarding
- Partner portals
- Performance reporting
- Renewal strategies
- Team and branch configuration
- Broader replacement of fragmented subscriptions
