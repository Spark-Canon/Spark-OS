# ADR-000: Repository Constitution

**Status:** Accepted  
**Date:** 2026-07-20

## Context

Spark-OS requires a durable source of architectural truth that survives conversation loss, personnel changes, and tool interruptions.

## Decision

Create `brain/constitution.md` as the canonical entry point and bootloader for the project.

## Consequences

- New collaborators have one clear starting point.
- The Constitution defines reading order and source-of-truth precedence.
- Changes to foundational project rules require explicit review and an ADR.
